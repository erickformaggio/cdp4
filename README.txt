#cdp4

INTRODUÇÃO

O Crawler Death proof 4 é uma atualização de sua primeira edição. Este script possibilita uma suíte de testes através de programas já instalados no Kali Linux.

SETUP

Caso você utilize outra distro, é necessário instalar:

TheHarvester
WPScan
Nikto
Nmap

EXEMPLO

Para utilizar, faça o download do script e execute-o através de sua própria pasta.

./crawler-death-proof4
